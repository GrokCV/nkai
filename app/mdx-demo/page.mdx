# MDX 示例页面

欢迎来到 MDX 示例页面！这个页面展示了如何在 Next.js 项目中使用 MDX。

## 什么是 MDX？

MDX 是一个允许你在 Markdown 文档中编写 JSX 的格式。它结合了 Markdown 的简洁性和 React 组件的强大功能。

### 嵌入 JSX 组件

你也可以在 MDX 中直接使用 JSX：

</div>

## 总结

MDX 提供了一种强大的方式来创建富文本内容，同时保持了 Markdown 的简洁性。你可以：

- 使用标准的 Markdown 语法
- 嵌入 React 组件
- 自定义组件样式
- 创建交互式文档
